星际译王（stardict）  
安装方法：在当前文件夹右键，打开终端，输入sudo sh install.sh.软件和当前文件夹中的词典一站式安装。  
如果当前文件夹中的词典包有变动，需要修改install.sh文件。修改细节参考后面的部分。
