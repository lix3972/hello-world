1、共享文件夹  
右键属性->权限都改为创建和删除文件，->本地网路共享，勾选共享此目录，允许其他人创建和删除……。
2、手机访问共享文件夹  
在手机上下载es文件浏览器。打开ES文件浏览器，点左上角三横道，选网络-局域网，第一次输入ubuntu系统用户名和密码，可以看到共享的文件夹，但是，不能点开。  
问题分析：共享文件夹可能没有访问权限。
3、修改访问权限  
参考https://jingyan.baidu.com/article/b24f6c826efc7f86bee5da46.html
ubuntu中如何进行文件权限设置  
3.1、在终端 输入命令：ll两个L。
r代表读，w代表写，x代表执行，例如：-rw-rw-r-- 1 python python 0 3月 30 15：58 2.py
rwx表示可读可写可执行，r-代表只读,rx 代表可读可执行但不可写。  
3.2用chmod命令，修改权限。  
可以给三个类别人员设置权限，三个类别为拥有者，同组者，其他人员。  
u代表拥有者，g代表同组者，o代表其他人员。
例如：chmod u=rwx 2.py 给拥有者设置权限(进入root权限，或sudo chmod u=rwx 2.py )     
sudo chmod u=r,g=r,o=r 2.py 给三类人员赋予操作文件2.py相应的权限。
